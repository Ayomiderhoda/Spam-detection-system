⚠️ IMPORTANT: You CANNOT open these HTML files directly
   in Edge or Chrome. You MUST run app.py first.
   Here is exactly what to do:

-----------------------------------------------------
STEP 1 — Install Python (if not already installed)
-----------------------------------------------------
  Download from: https://www.python.org/downloads/
  ✅ During install, tick "Add Python to PATH"

-----------------------------------------------------
STEP 2 — Open Command Prompt in this folder
-----------------------------------------------------
  - Open the final_project folder
  - Hold SHIFT + Right-click inside the folder
  - Click "Open PowerShell window here"
    OR
  - Search for "cmd" in Windows, open it, then type:
      cd C:\path\to\final_project

-----------------------------------------------------
STEP 3 — Install required packages (only once)
-----------------------------------------------------
  Type this and press Enter:

    pip install flask joblib scikit-learn

  Wait for it to finish.

-----------------------------------------------------
STEP 4 — Run the app
-----------------------------------------------------
  Type this and press Enter:

    python app.py

  You will see:
    * Running on http://127.0.0.1:5000

-----------------------------------------------------
STEP 5 — Open in Edge / Chrome
-----------------------------------------------------
  Open your browser and go to:

    http://127.0.0.1:5000

  That's it! The website will load.
